# Tic Tac Toe (Python)

โปรเจกต์นี้เป็นเกม Tic Tac Toe (OX) เขียนด้วยภาษา Python

## วิธีใช้งาน

1. ติดตั้ง Python (แนะนำ Python 3)
2. เปิดเทอร์มินัลหรือ Command Prompt
3. รันคำสั่งต่อไปนี้ในโฟลเดอร์นี้

```sh
python tictactoe.py
```

## คุณสมบัติ

- เล่นเกม Tic Tac Toe แบบ 2 ผู้เล่น
- แสดงผลลัพธ์บนคอนโซล
